# THE UGLY BRAND — DROP 000

Interactive concept store for **Drop 000: The Boring One**.

Live site: <https://stefangrasic04-boop.github.io/Grasic-project/>

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL printed by Vite.

## Included interactions

- anti-entry landing screen with a T-shirt that avoids the pointer;
- crumple-to-assembled garment load animation;
- draggable front/back product rotation and scroll-to-zoom inspection;
- animated inside label and individual edition number;
- responsive product, bag, concept checkout, and order-success flow;
- sold-out state at `/?soldout=1`;
- reduced-motion accessibility support.

The checkout is deliberately a prototype and does not process payment.
